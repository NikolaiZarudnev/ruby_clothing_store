# Clothing store

This is the clothing store with the ability to add and remove items

## Requirements:

* Ruby version: 3.0.2
* MySQL (https://www.mysql.com/)

## Install:
```
gem install bundler
bundle install
```
### MySQL install:
#### Linux:
```sudo apt-get install libmysqlclient-dev```
#### Mac OS:
```brew install mysql```
## Run:
```rails server```
### Use:
Enter in browser:

```http://127.0.0.1:3000/```
